# API de Gerenciamento de Currículos

Esta API permite que os usuários realizem o upload e download de currículos nos formatos PDF, DOC e DOCX. Ela foi desenvolvida como parte do curso diversedev da Ada tech, com o objetivo de treinar habilidades em desenvolvimento web com ASP.NET Core.

## Funcionalidades

- **Upload de Currículo**: Os usuários podem enviar seus currículos para a API.
- **Download de Currículo**: Os usuários podem baixar os currículos previamente enviados.

## Validações

- **Tamanho do Arquivo**: O tamanho máximo permitido para um currículo é de 5MB.
- **Extensão do Arquivo**: A API aceita apenas arquivos nos formatos PDF, DOC e DOCX.
